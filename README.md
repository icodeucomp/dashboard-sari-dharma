# Cpanel Deployment

Node Version : 22.14.0

1. Build Locally using npm run build
2. Copy all file except node_modules and unrelated deployment data
3. Setup Node application in Cpanel hosting
4. Paste in Cpanel hosting node applocation root directory
5. Configure environment variables in Cpanel node application interface
6. Install Dependecies using NPM install via Cpanel node application interface
7. Restart node application
